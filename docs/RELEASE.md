## Release

1. Create new release branch (e.g. `v0.4.8`)
2. Update `package.json` with new version (e.g. `v0.4.8`) in a PR
3. Once ready for release, merge branch PR to main
4. Create new Git tag and push (e.g. `v0.4.8`)
5. Confirm release is [available in NPM](https://www.npmjs.com/package/@pyroscope/nodejs?activeTab=versions)
